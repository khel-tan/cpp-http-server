Detailed Description

A server that provides HTTP request handling services.

HTTP Server handles management of resources required to transform an HTTP request string into a request object, pass said object to some handlers and obtain some response string, send back the above string over the network, and do error handling for all of the above.

Its dependencies are provided through Socket and RequestParser interfaces.
